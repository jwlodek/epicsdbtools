# epicsdbtools

[![CI](https://github.com/jwlodek/epicsdbtools/actions/workflows/ci.yaml/badge.svg)](https://github.com/jwlodek/epicsdbtools/actions/workflows/ci.yaml) [![Coverage](https://codecov.io/github/jwlodek/epicsdbtools/graph/badge.svg?token=7VS242V6D7)](https://codecov.io/github/jwlodek/epicsdbtools) [![License](https://img.shields.io/badge/license-BSD--3--Clause-blue?style=flat-square)](https://opensource.org/license/bsd-3-clause)

Python module that provides useful utilities for interacting with EPICS database templates and substitution files. It can parse both into python objects, and provides tools that can leverage this capability to generate code from these representations.
